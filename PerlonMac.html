. Macで脳波解析を行うためにPerl環境を整える

Macで脳波解析を行うためにPerl環境を整える

1. XCodeのインストール

App Storeで"Xcode"と検索し、ダウンロード・インストールして下さい（ファイルサイズが大きく、時間がかかります） [link]

2. homebrewのインストール

terminal.appを開き、以下のスクリプトを実行します（参考）

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

3. XQuartzのインストール

以下のページからXQuartzのdmgファイルをダウンロードし、インストールします（参考）

https://www.xquartz.org

4. SciPDLのインストール

以下のページからSciPDLのdmgファイルをダウンロードし、Applicationフォルダに入れます。

https://sourceforge.net/projects/pdl/files/SciPDL/

5. Perlのインストール状態の確認

terminal.appで以下を入力し、バージョン情報が帰ってくるかを確認

$ /Applications/PDL/bin/perl5.22.1 -v

いかのような出力が返って来ればOK。 

出力例：This is perl 5, version 22, subversion 1 (v5.22.1) built for darwin-2level

＊「perlは、開発元を検証できないため開けません」と表示された場合、以下の手順で許可を与えます。
  • 「システム環境設定」を開く
  • 「セキュリティとプライバシー」を開く
  • 下に表示されえている「変更するにはカギをクリックします」をクリックし、アカウントのパスワードを入力
  • 「ダウンロードしたアプリケーションの実行許可」の欄に「"perl5.22.1"は開発元を確認できないため、使用がブロックされました」と表示されているので、その右にある「このまま許可」をクリック

    • 6. パス（path）の設定

    terminal.appで以下を入力します。

    $env | grep PATH

    ＊ envの次は|です。iでもIでもありません。

    以下のようにパス一覧が表示されます。

    出力例：PATH=/opt/local/bin:/opt/local/sbin:/Users/my/anaconda3/bin .... （省略）

    ＊パスは、コロン（:）で区切られています。
    ＊シンボリックリンクの作成先は、使いやすい・覚えやすいところで良いですが、並びの早いところに入れたほうが良いです。パスは左に書いてあるほうが優先されるからです （参考）

    出力の中から一つ選び、perlのシンボリックリンク（シンボリックリンクとは）を作成します。 

    $sudo ln -s /Applications/PDL/bin/perl5.22.1 行先/perl
    入力例：$sudo ln -s /Applications/PDL/bin/perl5.22.1 /opt/local/bin/perl

    矢野メモ: /opt/local/bin/perlに作ったらFile existsという表示。ディレクトリを見てみると「種類：エイリアス」になっているperlが最初から存在しているようである。これを使えばシンボリックリンクの作成は必要ない？

    7. モジュールのインストール

    terminal.appで以下を実行

    $sudo perl -MCPAN -e shell

    矢野メモ：以下のメッセージが表示されたのでyesにした。
    CPAN.pm requires configuration, but most of it can be done automatically. If you answer 'no' below, you will enter an interactive dialog for each configuration option instead. Would you like to configure as much as possible automatically? [yes]

    ターミナルにcpan[1]>と表示されればOK。1つずつインストールをしていきます。

    cpan[1] > install Math::FFT
    cpan[2]>install PDL::DSP::Windows
    cpan[3]>install PDL::DSP::Fir
    cpan[4]>install Hash::Merge
    cpan[5]>install Graphics::GnuplotIF
    cpan[6]>install Term::ReadPassword
    cpan[7]>install Array::Uniq
    cpan[8]>install PGPLOT
    cpan[9]>install PDL::Graphics::Gnuplot

    Hash::Mergeを入れるときに次のモジュールも入るはずですが、入っていることを確認するために以下を実行して下さい 

    cpan[8]> install Test::Pod

    以下のような出力が返ってくればインストールされています。 

    cpan[8]> install Test::Pod
    　Test::Pod is up to date (1.52).
